Classes

Dans LeekScript, toutes les valeurs sont associées à une classe, il y a 8 classes de base :
  • Null : possède une unique valeur qui peut servir à représenter une variable indéfinie ou une erreur.
    Valeurs possible : null
  • Boolean : représente un booléen, avec 2 valeurs possibles : vrai ou faux.
    Valeurs possibles : true, false
  • Number : représente un nombre. Les nombres possèdent des méthodes comme .abs(), .cos() etc. détaillées plus loin.
    Valeurs possibles : 0, 12, -5, 3.141
  • String : représente une chaîne de caractères, un texte.
    Valeurs possibles : '', 'hello', '12345', 'i 4 _ 3a'
  • Array : un tableau (ou liste) contenant d'autres valeurs.
    Valeurs possibles : [], [1, 2, 3], [1, true, 'hello'], [[1, 2], 3]
  • Object : un object contient des valeurs, associées à une clé (chaîne de caractère).
    Valeurs possibles : {}, {a: 1, b: 2, c: 3}, {a: 1, b: true, c: 'hello'}, {a: {a: 12}}
  • Function : représente une fonction, qui prend en paramètre une ou plusieurs valeurs et en renvoie une seule. Il y a deux notations : celle avec le mot-clé 'function' et celle simplifiée avec une flèche.
    Valeurs possibles : function() {}, function(x, y) { return x + y }, -> null, x, y -> x + y
  • Class : Classe qui représente une classe.
    Valeurs possibles : Null, Boolean, Number, String, Array, Object, Function, Class, class Foo {}, class Bar { let x; let m = function() {} }
L'utilisateur peut créer ses propres classes, et ainsi crééer plus de classes : class Foo {}. Les classes créées dérivent de la classe de base Object par défaut. Mais il est possible de préciser de quelle classe de base dérive sa nouvelle classe : class Foo extends String {}.

Variables

Une variable est un emplacement mémoire associé à un nom, qui permet de stocker une valeur (valeur parmis les classes de base vu ci-dessus).
Exemples : let a = 2, let b = 'hello', let foo = [1, 2, 3].
